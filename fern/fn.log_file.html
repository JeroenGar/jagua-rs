pub fn log_file<P: AsRef<Path>>(path: P) -> Result<File>
Expand description

Convenience method for opening a log file with common options.

Equivalent to:

std::fs::OpenOptions::new()
    .write(true)
    .create(true)
    .append(true)
    .open("filename")

See OpenOptions for more information.
