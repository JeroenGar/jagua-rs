Expand description

Return an iterator of floats from a to b (exclusive), incrementing by step.

Numerical reasons can result in b being included in the result.

The iterator element type is F, where F must implement Float, e.g. f32 or f64.

Panics if converting ((b - a) / step).ceil() to type F fails.
