Enum NormalRotDistr

Source 
pub enum NormalRotDistr {
    Range(Normal<f32>),
    Discrete(f32),
    None,
}
Expand description

Samples a rotation from a normal distribution over a given range or a discrete set of rotations. In case of discrete rotations the mean is always returned.
