A fast and fearless Collision Detection Engine for 2D irregular cutting and packing problems.

This library is designed to be used as a backend by optimization algorithms.

  • spp — Enables support for the Strip Packing Problem
  • bpp — Enables support for the Bin Packing Problem
  • mspp — Enables support for the Multi Strip Packing Problem
